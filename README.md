# Hamster

A Web Framework,like rails.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'hamster', :git => git@github.com:sllt/hamster.git
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install hamster
